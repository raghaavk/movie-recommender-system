# 🎬 Movie Recommender System

## 📌 Project Description

This project is a Movie Recommendation System built using Python.
It recommends movies based on similarity between movies using rating data.

## 📂 Dataset

The project uses two datasets:

- **movies.csv** – contains movie titles and movie IDs
- **ratings.csv** – contains user ratings for movies

## ⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

## 🚀 How to Run the Project

1. Clone the repository
2. Install dependencies
3. Run the main program

Example:

pip install -r requirements.txt
python main.py

## 📊 Features

- Movie similarity recommendation
- Data preprocessing using Pandas
- Recommendation based on user ratings
